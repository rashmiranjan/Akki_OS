# AKKI OS 🚀

**Personal Branding Operating System**

AKKI OS is a 24x7 autonomous Personal Branding Operating System. Unlike chatbots that wait for commands, Akki proactively manages your entire personal brand - from research to publishing - while you sleep.

[![CI](https://github.com/Chiraggoyal120/mission_control/actions/workflows/ci.yml/badge.svg)](https://github.com/Chiraggoyal120/mission_control/actions/workflows/ci.yml)

---

## 🌟 Core Philosophy

- **Bot initiates, user approves** - Not the other way around.
- **Permanent memory** - The user never repeats themselves.
- **Silent learning** - Adapts from every edit, like, and comment.
- **Multi-agent brain** - Specialized agents for each specific task.
- **Zero chatbot feel** - Dashboard + Commands + Notifications.

## 🚀 Competitive Advantage

| **Feature** | **Existing Tools** | **AKKI OS** |
| --- | --- | --- |
| **Initiative** | User asks → Bot responds | Bot acts → User approves |
| **Memory** | Session / Short-term | Permanent Graph Memory |
| **Learning** | Explicit feedback only | Implicit (edits, engagement) |
| **Research** | Manual input required | Autonomous 24/7 Intel |
| **Engagement** | None | Auto reply suggestions |
| **Strategy** | Fixed calendar | Self-healing performance |

---

## 🏗️ System Architecture

| **Layer** | **Technology** | **Purpose** |
| --- | --- | --- |
| **Runtime Engine** | OpenClaw | Agent orchestration |
| **Dashboard** | Next.js + TS | Per-user control room |
| **Backend API** | Node.js + Express | Business logic + agent control |
| **Database** | Supabase (PostgreSQL) | User data + memory storage |
| **Bot Interface** | Telegram | Primary user interaction |
| **Deployment** | Docker + VPS | Production hosting |

## 🤖 The Agent Cabinet

AKKI OS operates with a specialized team of autonomous agents:

- **Jarvis (Orchestrator)**: Parses intent, delegates tasks.
- **Fury (Researcher)**: Autonomous web intel scraping.
- **Oracle (Idea Generator)**: Creates hundreds of content ideas from research.
- **Loki (Writer)**: Generates LinkedIn/X/Newsletter posts.
- **Atlas (Distributor)**: Schedules and publishes to platforms.
- **Echo (Engagement)**: Reply suggestions + conversation hunting.
- **Pulse (Analytics)**: Performance insights + auto-evolution.
- **Shuri (Strategist)**: 7-day content calendar + campaign planning.

## 🛠️ Get Started in Minutes

### 1. Configure Environment

```bash
cp .env.example .env
```

Set your Supabase and OpenClaw credentials in the `.env` file.

### 2. Start AKKI OS

```bash
docker compose -f compose.yml --env-file .env up -d --build
```

### 3. Access Your Control Room

- **AKKI Dashboard**: `http://localhost:3000`
- **Backend Health**: `http://localhost:8000/healthz`
